# 설명
마인크래프트 plugin.yml 파일

# 기능


# 사용법
```yaml
name: MiniGameMaker
main: com.wbm.minigamemaker.Main
version: 1.0
description: worldbiomusic (discord -> worldbiomusic#7075)
api-version: 1.16
depend: [wbmMC]

commands:
  minigame:
```


# 주의사항
- wbmMC 라이브러리가 항상 같은 폴더에 필요함

# 개선할 것
- command permission 알아보기
- api-version 17 변경
