# 1. Sign
- Must set `minigame-sign` in `settings.yml` to true  
<img src="sign-join.png" width="49%"></img>
<img src="sign-leave.png" width="49%"></img>
## Join
```yaml
[MiniGame]
 <title>
```
## Leave
```yaml
[Leave Minigame]
```


# 2. Command
- Join: `/minigame join <title>`
- Leave: `/minigame leave`

# 3. MENU
- `/minigame menu`
- Join: click minigame
- Leave: click leave icon
